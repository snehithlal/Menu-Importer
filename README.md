# Menu-Importer

Simple application to import menu groups and items for restaurants

## Requirement

- Ruby - 3.3.0
- Rails -6.1
- Postgresql

## Installation

- clone
- bundle
- cp config/database.yml.example config/database.yml
- yarn add jquery popper.js semantic-ui-sass
- rake db:create
- rake db:migrate
- rake db:seed
